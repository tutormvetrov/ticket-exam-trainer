# Roadmap

## Уже закрыто

1. Слоистая структура проекта.
2. SQLite persistence и import pipeline.
3. Main exam flow на Windows.
4. Профиль `Госэкзамен`.
5. `Dialogue` flow.
6. DLC `Подготовка к защите`.
7. Разделение `ui/theme.py` на пакет `ui/theme/`.
8. Warm-minimal visual refresh этапа 1:
   - shell;
   - `Библиотека`;
   - `Билеты`;
   - `Тренировка`;
   - visual gate screenshots;
   - light/dark click audit.
9. Отдельный `review` workspace в тренировке.

## В активном горизонте

1. Visual pass для остальных экранов:
   - `Предметы`
   - `Разделы`
   - `Импорт`
   - `Диалог`
   - `Статистика`
   - `Карта знаний`
   - `Подготовка к защите`
   - `Настройки`
2. Полноценный `import preview`.
3. Дополнительный polish вокруг large data states и onboarding.
4. Реальный runtime smoke на macOS.

## Что не стоит смешивать с текущим этапом

- новые облачные интеграции;
- полный billing-product вокруг DLC;
- крупный backend rewrite без пользовательской причины.
