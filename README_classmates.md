# Тезис

Короткая инструкция для однокурсников. Это локальное приложение для подготовки к письменному госэкзамену ГМУ. Внутри уже есть 208 билетов. Интернет нужен только если захотите подключить `Ollama`.

## Что лежит в архиве

- `Tezis.exe` на Windows или `Tezis.app` на macOS
- папка `scripts` для диагностики и ручных сценариев
- папка `data`
- этот файл

## Как начать

1. Распакуйте архив целиком в обычную папку. Не запускайте приложение прямо из zip.
2. Запустите `Tezis.exe` или `Tezis.app`.
3. При первом запуске создайте профиль. Нужны только имя и аватар.
4. Приложение откроет `Дневник`. Это ваш главный экран на каждый день.
5. Перейдите в `Билеты`, найдите нужный билет по номеру, словам, разделу или сложности.
6. Откройте тренировку и идите по режимам в таком порядке:
   `Чтение эталона` → `План` → `Пропуски` → `Активное воспроизведение` → `Полный госответ` → `Рецензия`.
7. Если хотите более подробную рецензию, откройте `Настройки` и подключите `Ollama`.

## Что важно знать

- По умолчанию приложение открывается на весь экран. `Esc` переводит его в оконный режим.
- В приложении нет импорта документов. Все 208 билетов уже встроены.
- Весь прогресс хранится локально.

Windows:
`%LOCALAPPDATA%\Tezis\app_data\`

macOS:
`~/Library/Application Support/Tezis/app_data/`

## Если Windows показывает SmartScreen

Нажмите `Подробнее`, потом `Выполнить в любом случае`.

## Если macOS блокирует запуск

Нажмите правой кнопкой по `Tezis.app` и выберите `Открыть`.

## Если рецензия слишком простая

Скорее всего, полная рецензия ещё не настроена. Откройте `Настройки` → `Рецензирование`.

- Если видите `Скачать Ollama for Mac`, сначала установите Ollama.
- Если видите `Запустить Ollama`, дайте приложению поднять локальный сервис.
- Если видите `Скачать qwen3:8b`, загрузите модель прямо из приложения.

Папка `scripts` в архиве нужна только для ручной диагностики, если стандартный сценарий из приложения не помог.

## Если нужно перенести прогресс

Скопируйте папку `app_data` на другой компьютер в тот же путь.
